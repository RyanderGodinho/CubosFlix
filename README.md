# Aplicação para um serviço de streaming (pense num Netflix).

Um website que permite:

- Visualização de filmes 
- Paginação de filmes 
- Busca de filmes 
- "Filme do dia" 
- Modal de filme 
- Mudança de tema

Os dados do website forão requisitados diretamente de uma API.
